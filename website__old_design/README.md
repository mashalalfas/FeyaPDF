# Archived — Old Feya PDF Website Design

This directory contains the source files for the original **"Apprentice's Desk"** design — a warm paper-toned, 7-section landing page with Three.js desk scene, flipbook, ink palette, wax seal, ember animations, GSAP scroll transitions, and Lenis smooth scroll.

This design was deployed to `gh-pages` but was **not the correct design**. The correct minimal dark design lives in the HERO repo:

- **Source:** `mashalalfas/hero/skills/web-designer/feya-website/index.html`
- **Live:** https://mashalalfas.github.io/FeyaPDF/

## What's here

| File | Description |
|------|-------------|
| `src/style.css` | CSS custom properties, desk scene, animations, HUD |
| `src/main.js` | Lenis, GSAP ScrollTrigger, interactions, easter eggs |
| `src/scene.js` | Three.js low-poly study desk with lamp, notebook, plant |

These were inlined into a single `index.html` for GitHub Pages compatibility and pushed to `gh-pages`. The commit history still contains the full design if needed.
